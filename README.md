# Adyant's Hackpad
Hi! My hackpad is a 4 key macropad designed to help my productivity in fusion 360 and my day-to-day life. 
## Features:
- 4 switches
- 1 rp2040
- 2 leds
- Key's designed to help me make the most out of fusion 360 like the extrude key (e)
- Other key's designed for mac-OS optimized shortcuts
- Uses KMK firmware and can be easily remapped
## Inspiration: 
While working between Fusion 360, coding environments, and general macOS tasks, I noticed that a small number of actions were repeated constantly. That's why I built this Hackpad to speed up common actions I use constantly across macOS and Fusion 360, such as undo, extrude, and app switching.
